## Competition project
Prototype project for code competitions

# Manual: build and run
    cd src/main/java
    javac Main.java
    java Main <../../../example.in >../../../example.out

# Gradle: test
    ./gradlew test

# Gradle: Build and run
    ./gradlew assemble
    cd build/classes/main
    java Main <../../../example.in >../../../example.out
